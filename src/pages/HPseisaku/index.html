ホームページを作ってみましょう

HTMLとCSSを使って、実際にサイトを作ってみましょう

  • ホーム
  • HTMLの土台
  • CSSでデザイン
  • 画像を入れよう
  • リンクを貼る
  • フォームを作る
  • サンプルページ

1. ページの土台を作る

まずはHTMLでページの土台を作りましょう。下のように「見出し」と「文章」を書いてみます。

<h1>ようこそ！</h1>
<p>これははじめて作ったホームページです。</p>

2. デザインしてみよう

次にCSSを使って色やレイアウトを整えます。以下のようなコードをstyle.cssに書いてみましょう。

body {
  background-color: #f0f8ff;
  font-family: sans-serif;
  padding: 20px;
}

3. 画像を入れてみよう

画像は見た目の印象を大きく変える要素です。好きな写真やイラストを入れてみましょう。

<img src="images/my-photo.jpg" alt="自分の写真">

4. リンクを貼ろう

自分のSNSや別のページにつながるリンクをつけてみましょう。

<a href="https://example.com">わたしのページへ</a>

実例：わたしのページへ
