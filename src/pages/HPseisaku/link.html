1. リンクってなに？

リンクとは、クリックすることで別のページやサイトに移動できる「つながり」のことです。

たとえば、自分のSNSや他のサイトを紹介するときに使います。

2. 基本の書き方

リンクを作るには、<a> タグを使います。

<a href="https://example.com">こちらをクリック</a>

href 属性に「ジャンプ先のURL」を書きます。

実際の表示例：こちらをクリック

3. 新しいタブで開くには？

自分のページから外部サイトへ飛ばすときは、別タブで開いた方が親切です。

<a href="https://example.com" target="_blank">別タブで開く</a>

表示例：別タブで開く

4. 自分のページ内へのリンク

自分のサイトの別ページにもリンクできます。

<a href="profile.html">プロフィールへ</a>

「profile.html」というページが同じフォルダにあれば、そのページに移動します。

5. よくあるミスと注意点

  • リンク先のURLに間違いがあると移動できません
  • target="_blank" を忘れると、現在のページが上書きされる
  • ファイル名は Link.html と link.html は別物（大文字小文字に注意）
