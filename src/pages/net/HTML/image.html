1. 画像の基本

HTMLで画像を表示するには、<img> タグを使います。

たとえば、次のように書きます：

<img src="images/sample.jpg" alt="サンプル画像">

src（ソース）は画像のファイルパス、alt は「画像が表示されなかったときの説明文」です。

3. よく使う画像サイズ調整

CSSで画像のサイズを変えることもできます。

img {
  width: 300px;
  border-radius: 10px;
}

これで、画像の横幅を 300ピクセル にし、角を丸くできます。

4. 画像を使うときの注意点

  • 画像ファイルは必ず同じフォルダ内にあるか、正しいパスで指定しましょう
  • alt属性はアクセシビリティの観点でも重要です（音声読み上げにも対応）
  • ファイル名は sample.JPG と sample.jpg は違うと認識されます（大文字小文字に注意）
