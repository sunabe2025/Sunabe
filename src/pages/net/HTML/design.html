1. CSSってなに？

CSS（シーエスエス）は、HTMLで作られたページの「見た目」をきれいに整えるための言語です。

たとえば：

  • 文字の色や大きさを変える
  • 背景に色をつける
  • 写真やボタンの位置を整える

などができます。つまり、CSSは「洋服」のような役割です。

2. 基本のCSSを書いてみよう

以下のようにCSSを書くと、ページ全体の背景色や文字の見た目が変わります。

body {
  background-color: #f0f8ff;
  font-family: sans-serif;
  color: #333;
  padding: 20px;
}

この例では：

  • background-color は背景色を指定
  • font-family は文字の種類
  • color は文字の色
  • padding は余白

3. HTMLにCSSを適用する方法

HTMLの中でCSSファイルを読み込むには、<head>タグの中に次のように書きます：

<link rel="stylesheet" href="style.css">

style.css は、CSSの内容を書いたファイル名です。

このようにして、HTMLとCSSを分けて管理するのが一般的です。
