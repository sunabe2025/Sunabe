Step 1：自己紹介ページを作ってみましょう

以下のような自己紹介ページを、HTMLの基本構造を使って作ってみましょう。

<!DOCTYPE html>
<html lang=\"ja\">
  <head>
    <meta charset=\"UTF-8\">
    <title>自己紹介</title>
  </head>
  <body>
    <h1>自己紹介</h1>
    <p>こんにちは！〇〇です。</p>
    <img src=\"images/my-photo.jpg\" alt=\"自分の写真\">
    <h2>すきなこと</h2>
    <ul>
      <li>すし</li>
      <li>どうぶつ</li>
      <li>ゲーム</li>
    </ul>
  </body>
</html>

Step 2：CSSで見た目を整えてみましょう

上で作成したHTMLにスタイルを加えることで、ページの見た目が整います。

<style>
  body {
    background-color: #fff8dc;
    text-align: center;
    font-family: sans-serif;
  }
  img {
    width: 200px;
    border-radius: 50%;
    margin-top: 20px;
  }
  ul {
    list-style: none;
    padding: 0;
  }
  li {
    margin: 5px 0;
  }
</style>
